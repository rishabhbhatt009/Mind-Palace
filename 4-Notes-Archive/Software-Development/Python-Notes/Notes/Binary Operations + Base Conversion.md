## Binary Operation

```python
### convert to int binary 
decimal = 10
binary = bin(decimal) # = 'xb1010' this can be converted back using int(binary,2)

### convert bin to int 
binary = '11'
dec = int(binary,2) # = 3

### xor 
a = 1
b = 2
c = a ^ b # = 3
```
