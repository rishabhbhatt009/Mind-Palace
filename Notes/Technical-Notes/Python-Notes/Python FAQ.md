- `heapq`
- `map()` : returns an iterable
- `''.join()` : 
	- can take iterables as arguments
	- elements should be `string` not `int`

	- `print('$$$'.split('$'))` -> `['', '', '', '']`
- reversing a string/list -> `lst[::-1]` = `lst[-1::-1]` = `lst[-1:-len(lst)-1:-1]`
